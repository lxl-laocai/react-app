## git 初始化

```git
  git add .   // 添加项目所有文件到本地仓库
  git commit -m "提交消息" // 版本提交和备注
  git remote add origin 仓库地址 // 本地仓库和git仓库进行关联
  git branch -M master // 创建分支
  git push origin master  // pull本地仓库的内容
```
